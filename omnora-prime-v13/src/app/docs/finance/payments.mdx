# Recording Payments

**Persona**: Accounts Manager

Update client balances and record cash/digital inflows.

### Prerequisites
- Outstanding invoice or client balance in the ledger.

### Step-by-Step Instructions
1. Navigate to **Finance > Payments**.
2. Select the **Party Name** (Customer or Supplier).
3. Enter the **Payment Amount** (e.g., Rs. 50,000.00).
4. Select the **Payment Method** (Cash, JazzCash, Bank Transfer).
5. Attach a reference number or receipt photo if required.
6. Click **Record Payment**.

### Troubleshooting
- **Unallocated Payment**: If not linked to an invoice, the amount will be added as "Advance Credit" to the party's Khata.
- **Wrong Date**: Use the **Ledger Correction** tool to modify transaction timestamps.
