# Data Sync Issues

**Persona**: IT Support / Manager

Resolve data synchronization lags between multi-location branches or cloud-backup endpoints.

### Prerequisites
- Active internet connection (if using Cloud Sync).
- Multi-location license (Elite).

### Step-by-Step Instructions
1. Navigate to **System > Sync Status**.
2. Identify which **Branch Node** is currently offline or lagging.
3. Click **Force Sync** to trigger a manual delta update.
4. Check the **Sync Logs** for any "Conflict Detected" flags.
5. In case of conflicts, the Hub will prioritize the **Timestamp** of the latest modification.

### Troubleshooting
- **Incomplete Sync**: If your internet is intermittent, the sync engine will pause and resume automatically. Do not close the Hub during a critical sync operation.
- **Database Locked**: Ensure no other process is accessing the `noxis.db` file directly.
