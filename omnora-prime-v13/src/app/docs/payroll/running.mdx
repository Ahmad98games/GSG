# Running Payroll Period

**Persona**: Accounts Manager

Execute the weekly or monthly payroll cycle and generate worker payouts.

### Prerequisites
- Production data synchronized for piece-rate workers.
- Attendance logs completed for the period.

### Step-by-Step Instructions
1. Navigate to **Workforce > Run Payroll**.
2. Select the **Billing Period** (Start and End Dates).
3. Review the **Gross Earnings** calculated from production units and attendance.
4. Verify **Peshgi (Advance)** deductions.
5. Click **Process All Slips**.
6. Select **Send Slips** to dispatch via WhatsApp and Print.

### Troubleshooting
- **Incorrect Units**: Re-sync the production database if piece-rate counts do not match the physical floor logs.
- **Deduction Mismatch**: Verify advance settlements against the client Khata for that specific worker.
