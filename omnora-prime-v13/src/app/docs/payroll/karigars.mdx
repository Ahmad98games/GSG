# Setting up Karigars

**Persona**: HR Manager / Production Supervisor

Register artisans and industrial workers for payroll and production tracking.

### Prerequisites
- Workforce module enabled in license tier.

### Visual Workflow
> [DIAGRAM]: Register Profile -> Assign Wage Type -> Verify WhatsApp -> Active Status

### Step-by-Step Instructions
1. Navigate to **Workforce > Artisans**.
2. Click **Add New Karigar**.
3. Enter the worker's **Full Name** and **Worker Code** (e.g., K-001).
4. Define the **Wage Type**: Piece-rate or Daily/Monthly fixed.
5. Enter the **WhatsApp Number** for automated digital pay slips.
6. Click **Save Profile**.

### Example Configuration
> [EXAMPLE]:
> **Name**: Muhammad Ahmed
> **Code**: K-402
> **Wage**: Piece-rate (Rs. 45/unit)
> **Phone**: +92 300 1234567

### Troubleshooting
- **WhatsApp Notification Failure**: Ensure the mobile number is entered in international format (e.g., 923xxxxxxxxx).
- **Inconsistent Rates**: Verify that the piece-rate logic matches the current production module configuration.
