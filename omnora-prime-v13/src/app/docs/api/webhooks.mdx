# Webhook Events

**Persona**: Developer

Integrate third-party services with the Noxis Hub using reactive webhooks.

### Prerequisites
- External server capable of receiving HTTP POST requests.
- Developer API key generated in Hub Settings.

### Step-by-Step Instructions
1. Navigate to **Developer > Webhooks**.
2. Click **Create New Endpoint**.
3. Enter your **Target URL** (e.g., `https://your-server.com/api/noxis-events`).
4. Select the events you wish to subscribe to:
   - `inventory.low_stock`
   - `finance.payment_received`
   - `security.alert_triggered`
5. Click **Test Ping** to verify the connection.

### Troubleshooting
- **403 Forbidden**: Ensure your server accepts requests from the Hub's IP address.
- **Payload Mismatch**: Verify your parser handles the Noxis JSON schema correctly.
