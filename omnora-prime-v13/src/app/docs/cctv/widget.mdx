# Sentinel Live Security Widget

The **Sentinel Live Widget** provides a real-time security overview directly on your dashboard, combining live MJPEG streams with AI-driven motion detection alerts.

## Key Features

### 1. Multi-Node MJPEG Streaming
The widget automatically cycles through or displays the most critical cameras configured in your mesh. It supports standard industrial MJPEG streams served via the local node proxy.

### 2. AI Detection Alert Flashing
When the **Sentinel AI Engine** detects movement or a security anomaly (e.g., unauthorized person in a restricted zone), the dashboard widget provides instant visual feedback:
- **Red Border Flash**: The entire widget flashes red for 300ms to grab the operator's attention.
- **Alert Overlay**: A floating red banner appears at the bottom of the widget indicating the specific camera and event.
- **Detection Counter**: A real-time counter tracks total detections for the current operational shift.

## Configuration

### Status Indicators
- **Live (Green Pulse)**: The feed is active and healthy.
- **Offline (Gray)**: The camera node is unreachable or the stream has timed out.

### Adding Cameras to Dashboard
To show a camera in the dashboard widget:
1. Go to **CCTV Sentinel** module.
2. Mark the camera as **"Priority Node"**.
3. Ensure the node status is set to **"Online"**.

## Security Protocol

The widget uses a secure HMAC-signed stream request to ensure that local camera feeds are only accessible within the authorized Noxis Hub environment.

> [!NOTE]
> For optimal performance, we recommend limiting the dashboard widget to 2 concurrent streams to preserve local network bandwidth. Full multi-view is available in the dedicated CCTV module.
