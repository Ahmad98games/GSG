# Understanding Alerts

**Persona**: Security Operator / Manager

Manage and acknowledge AI-driven security alerts generated by Sentinel.

### Prerequisites
- Active CCTV nodes with motion detection enabled.

### Step-by-Step Instructions
1. Monitor the **Dashboard Alert Feed** for real-time security events.
2. When a **Critical Alert** (Red) occurs, the screen will overlay with a forensic confirmation interface.
3. Observe the **Alert Type**: Human Detection, Area Intrusion, or Camera Tampering.
4. To acknowledge, type the **Node Label** into the confirmation box.
5. Review the **Security Log** later for audit trail analysis.

### Troubleshooting
- **False Alarms**: Lower the **Detection Sensitivity** in camera settings to filter out small animals or lighting changes.
- **Missing Notifications**: Ensure the **Sentinel Vision Service** is running in the background.
