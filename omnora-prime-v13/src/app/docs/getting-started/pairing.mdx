# Mobile Pairing Guide

**Persona**: Operator / Floor Manager

Establish a secure link between the Noxis Hub and a mobile device for floor operations.

### Prerequisites
- Noxis Hub installed and active on the primary workstation.
- Noxis Mobile app installed on an Android/iOS device.
- Both devices must be on the same local area network (LAN).

### Step-by-Step Instructions
1. Open the **Noxis Hub** on your desktop.
2. Navigate to **Settings > Mobile Pairing**.
3. A 6-digit pairing code will be displayed on the screen.
4. Open the **Noxis Mobile** app on your device.
5. Enter the 6-digit code when prompted.
6. Once validated, the device will appear in your **Paired Devices** list.

### Troubleshooting
- **Code Rejected**: Ensure the Hub is not already at its device limit (Lite: 5, Pro: 15).
- **Device Not Found**: Verify that the Hub is broadcasting on the correct network interface.
- **Connection Timed Out**: Disable aggressive Windows Firewall rules that may block TCP/IP traffic on port 8080.
