# Mobile Device Pairing Protocol

Connect your field devices to the Noxis Hub for real-time inventory and production logging.

## Requirements
- **Noxis Mobile App** (v4.2+) installed on Android or iOS.
- **WiFi Connectivity**: Both the Hub and the mobile device must be on the same subnet.
- **Camera Access**: Required for scanning the secure pairing vector (QR Code).

## Pairing Procedure

1. **Access Pairing Terminal**
   On the Hub Dashboard, navigate to **Settings > Mobile Pairing**. This will generate a unique, time-bound pairing token.

2. **Open Scanner**
   Launch the Noxis app on your mobile device. Select **"Pair New Node"**.

3. **Verify Encryption Handshake**
   Point the mobile camera at the Hub screen. 
   [DIAGRAM]: Scan QR -> RSA Key Exchange -> Secure Session Established

4. **Assign Persona**
   Once paired, select the device's primary persona (e.g., *Inventory Manager*, *Floor Supervisor*, or *Karigar*).

5. **Test Communication**
   Trigger a "Ping" from the mobile app. The Hub console should log: `NSP_NODE_CONNECTED: [DEVICE_ID]`.

## Security Best Practices
- **Revoke Lost Devices**: If a device is lost, immediately revoke its token from the Hub Management panel.
- **Biometric Lock**: Enable biometric authentication on all mobile devices to protect the local encryption keys.
- **MAC Filtering**: For high-security zones, restrict Hub access to known device MAC addresses.
