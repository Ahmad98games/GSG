# Creating Branches

**Persona**: Business Owner / Administrator

Scale your industrial operation by adding additional factory locations or outlets (Elite Only).

### Prerequisites
- Active **Elite** Tier license.
- Network connectivity for cross-branch data aggregation (optional but recommended).

### Step-by-Step Instructions
1. Navigate to **System Administration > Site Management**.
2. Click **Initialize New Branch**.
3. Enter the **Site Name** and **Physical Address**.
4. Configure the **Primary Currency** and **Tax Jurisdiction** for this location.
5. Define the **Local Server IP** for local mesh node synchronization.
6. Click **Deploy Branch Configuration**.

### Troubleshooting
- **License Incompatibility**: Upgrading to Elite is required to add more than one branch.
- **Sync Failure**: Verify that the primary branch's port 443 is accessible if using cloud-relay synchronization.
